TitanBag Sync server facilitates secure cloud synchronizations, invitation-based partner connections, and offline-first client journal transactions. All synchronization datasets are authenticated and signed using JSON Web Tokens (JWT) to secure user privacy.

Endpoint Explorer

POST /register Register a new cloud profile

Creates a cloud profile on TitanBag, generates bcrypt password hashes, and returns a 30-day JWT session alongside a unique invitation code for partner syncing.

{
  "username": "alex_johnson",
  "email": "alex@example.com",
  "password": "strongpassword123",
  "display_name": "Alex"
}
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "user": {
    "id": "e4588eeb-4b2a-4aeb-9fd5-c9f2b8495fb6",
    "username": "alex_johnson",
    "email": "alex@example.com",
    "display_name": "Alex",
    "partner_share_code": "EXP-B9E8-3C2E-4D78-1A90",
    "created_at": "2026-07-14T16:15:30.000Z",
    "updated_at": "2026-07-14T16:15:30.000Z"
  }
}
curl -X POST http://localhost:5000/register \
  -H "Content-Type: application/json" \
  -d '{"username":"alex_johnson","email":"alex@example.com","password":"strongpassword123","display_name":"Alex"}'
POST /login Authenticate credentials & session

Accepts either username or email and validates password. Returns a JWT token and user profile metadata.

GET /profile Retrieve user profile metadata

Retrieves the active user credentials and details of the connected partner account if linked.

Authorization: Bearer 
{
  "user": {
    "id": "e4588eeb-4b2a-4aeb-9fd5-c9f2b8495fb6",
    "username": "alex_johnson",
    "email": "alex@example.com",
    "display_name": "Alex",
    "partner_share_code": "EXP-B9E8-3C2E-4D78-1A90",
    "created_at": "2026-07-14T16:15:30.000Z",
    "updated_at": "2026-07-14T16:15:30.000Z",
    "partner": {
      "id": "f8a002eb-012b-3fac-2ed3-124facf9001b",
      "username": "partner_display",
      "display_name": "Jamie"
    }
  }
}
curl -X GET http://localhost:5000/profile \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
POST /partner/connect Link accounts via invitation code

Connects the current account to a partner profile using the partner's unique share code (e.g. EXP-XXXX-XXXX-XXXX-XXXX). Binds data streams immediately.

// Headers: Authorization: Bearer 
{
  "partner_share_code": "EXP-B9E8-3C2E-4D78-1A90"
}
{
  "message": "Connected with partner successfully",
  "partner": {
    "id": "e4588eeb-4b2a-4aeb-9fd5-c9f2b8495fb6",
    "username": "alex_johnson",
    "display_name": "Alex"
  }
}
curl -X POST http://localhost:5000/partner/connect \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"partner_share_code":"EXP-B9E8-3C2E-4D78-1A90"}'
DELETE /partner/disconnect Unlink connected partner status

Severs the current active link between the user's account and their partner. Both users return to standalone states.

Authorization: Bearer 
{
  "message": "Partner disconnected successfully"
}
curl -X DELETE http://localhost:5000/partner/disconnect \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
POST /sync Synchronize database registers

Offline-first cloud replication interface. Uploads local journals modifying, deleting, or inserting values, and pulls down all modifications done by partner accounts since the last timestamp.

{
  "last_synced_at": "2026-07-14T10:00:00.000Z",
  "local_journals": [
    {
      "id": "a7b309f4-124b-48aa-b3e1-d246cf901111",
      "title": "Grocery Shopping",
      "amount": 42.50,
      "category": "Food",
      "notes": "Weekly stock up",
      "payment_method": "Card",
      "date": "2026-07-14T11:30:00.000Z",
      "updated_at": "2026-07-14T11:32:00.000Z",
      "deleted": false
    }
  ]
}
{
  "synced_ids": [
    "a7b309f4-124b-48aa-b3e1-d246cf901111"
  ],
  "remote_journals": [
    {
      "id": "c8d410f5-235c-59bb-c4f2-e357df012222",
